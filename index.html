Application Overview

Budget Buddy is a comprehensive expense tracker and budget planner designed for users to monitor their spending, set financial goals, and stay on budget. With user-friendly interfaces and real-time feedback, users can easily enter expenses, manage their budget, and receive insights to optimize their financial health.

Technology Overview

  • Languages: HTML, CSS, JavaScript, Node.js
  • Libraries/Frameworks: Express
  • Platforms: Web (responsive design for mobile and desktop)
  • Hosting: Microsoft Azure
